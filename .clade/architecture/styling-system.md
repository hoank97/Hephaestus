# Styling System

## Principles
- Minimal, readable UI
- Responsive by default
- Keep client-side JS minimal

## Tailwind usage
- Ưu tiên utility classes trong component.
- Tránh custom CSS nếu không cần.

## Typography
- Content dài nên dùng class `app-prose` để đồng bộ màu/chữ/khoảng cách.
- Code blocks dùng Shiki/themed styles theo cấu hình project.

## Theme
- Hỗ trợ light/dark mode từ header theme toggle.
