# Tech Stack

- Framework: **Astro**
- Styling: **Tailwind CSS**
- Language: **TypeScript**
- Content: **Markdown/MDX** + Astro Content Collections
- Search indexing: **Pagefind**

## Build flow
- Dev: `npm run dev`
- Build: `npm run build`

## Why this stack
- Astro phù hợp static-first, SEO tốt.
- Tailwind giúp phát triển UI nhanh, nhất quán.
- Content collections + zod đảm bảo frontmatter type-safe.
